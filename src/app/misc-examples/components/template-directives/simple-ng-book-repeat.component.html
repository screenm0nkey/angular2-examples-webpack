Custom *ngBookRepeat template using IterableDiffer

ngDoCheck and IterableDiffers, KeyValueDiffers Assign a structural directive dynamic content

In the template that is generated, we’re going to have a local view variable #peep, that will receive the value from the $implicit local variable. That’s the name of the local variable that Angular uses when “de-sugaring” the syntax into a template. 

<li *ngBookRepeat="let peep of people">
gets converted to 
<template ngBookRepeat [ngBookRepeatOf]="people" let-peep="$implicit"></template>
Here in the code is where we set the $implicit value to the change.item. change.item will be equal to the value of peep. 
let view = this.viewContainer.createEmbeddedView(this.template, {'$implicit': change.item});
