One great solution is to take advantage of the selector in the @Directive decorator.
You simply define a directive that selects for <li> elements, then use a
@ContentChildren query to filter all <li> elements down to only those that
are content children of the component.

Note:  It seems like it might work to select for only <li> elements
within <my-list> tags (e.g. "my-list li"), but it's important to note
that parent-child selectors aren't yet supported.  If you want to
limit the results to children of your component, using queries to filter is the best way.
